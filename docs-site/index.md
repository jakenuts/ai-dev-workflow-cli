# AI Dev Workflow CLI

A framework-agnostic tool that provides AI-guided development workflows and patterns.

## Quick Start

```bash
# Install globally
npm install -g ai-dev-workflow-cli

# Initialize in your project
ai-dev init

# Get help
ai-dev --help
```
